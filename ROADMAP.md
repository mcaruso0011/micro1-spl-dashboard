# Roadmap — micro1 SPL Dashboard

Features to build, roughly in order of impact. Use Claude Code to tackle these one at a time.

---

## Phase 1: Polish & Interactivity

- [ ] **Column sorting** — Click column headers to sort expert table
- [ ] **Expert sparklines** — Mini trend charts for each expert's last 7 days
- [ ] **CSV export** — Download expert data as CSV
- [ ] **Theme toggle** — Dark/light mode switch
- [ ] **Responsive layout** — Make it work well on tablet/mobile

## Phase 2: Deeper Operational Features

- [ ] **Project switcher** — Toggle between multiple simulated lab projects
- [ ] **Expert assignment workflow** — Drag-and-drop experts between pipeline stages
- [ ] **Quality drill-down** — Click an accuracy score to see task-level breakdown
- [ ] **Cost per task tracking** — Show cost metrics alongside quality/speed
- [ ] **SLA tracker** — Visualize client delivery timelines and SLA compliance

## Phase 3: Intelligence Layer

- [ ] **Anomaly detection** — Auto-flag when an expert's performance deviates from their baseline
- [ ] **Capacity planning** — Show recommended team size based on incoming task volume
- [ ] **Rhea AI QC simulator** — Interactive demo of how automated QC catches structural errors
- [ ] **Expert matching** — Given a task domain, suggest the optimal expert based on history

## Phase 4: Wow Factor

- [ ] **Real-time simulation** — Animated tasks flowing through the pipeline stages
- [ ] **Anthropic API integration** — Use Claude to generate natural-language project summaries
- [ ] **Interactive onboarding flow** — Walk a new SPL through the dashboard with guided tooltips
- [ ] **Comparative analytics** — Side-by-side project performance comparison
